Loading a Site Map

Note: This step is optional.

A site map contained in a "gif" or "jpg" graphics file can be loaded to the preprocessor to guide the drawing of graphical data.

To load a site map:

1. Select "Site Map" from the "Active Data" Selector (see ).

2. Click the  button. The "Load Site Map" dialog box is displayed.

3. Navigate to the directory or folder containing the site map file, and select the site map file.

4. Click the "Open" button. The site map is displayed at the upper left corner of the drawing area.

After loading the site map, you may:
 •  Move the site map
 •  Resize the site map
 •  Resize the site map about a fixed point
 •  Delete the site map
