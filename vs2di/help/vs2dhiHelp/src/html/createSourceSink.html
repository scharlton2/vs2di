Create Source/Sink Points

Before you begin, make sure that "Source/Sink Points" is selected in the "Active Data" Selector (see ).

1. Click the  button.

2. Move the cursor to the desired location for the source/sink point and click the left mouse button.

3. A dialog box appears for you to set the flow rate and temperature. Initially, the flow rate and temperature are both set to zero for all recharge periods. To proceed, see the Source/Sink Properties Dialog Box.
