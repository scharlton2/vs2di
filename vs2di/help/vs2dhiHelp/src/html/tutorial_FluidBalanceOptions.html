Tutorial: Fluid Balance Options

In the fluid balance panel, mass balance components for flow are selected to be written to auxiliary output file "balance.out". For this example, 4 selections are made: Total Fluid In and Out of the Domain, Change in Fluid Storage, and Fluid Balance.

After completion of the simulation, "balance.out" will list 3 values for each of these selections (Total for Simulation, Total for Time Step, and Rate for Time Step) for every time step in a format that is suitable for input to standard plotting programs.
