Copying Boundary Conditions

You may copy all boundary conditions from one recharge period to other recharge periods. This option is available only when there are two or more recharge periods.

Before you begin, make sure that "Boundary Condition" is selected in the "Active Data" Selector (see ). If the Recharge Period Window is not visible, click the "Show" menu and select "Recharge Period Window".

1. In the Recharge Period Window, click the row displaying the recharge period of the boundary conditions you wish to copy.

2. Click the "Edit" menu and select "Copy all to..." The "Copy BC" dialog box is displayed.

3. Select (click) the recharge period to which you want to copy the boundary conditions. To select multiple periods, hold down the "Ctrl" key and click each period.

4. Click "OK".
