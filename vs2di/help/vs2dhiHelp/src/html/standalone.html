The Standalone Postprocessor

The Postprocessor may be launched as a standalone application to:

• Play back a saved simulation. (This is the main intended use of the standalone postprocessor.)
• Run a simulation using data files exported from the preprocessor, and possibly modified by the user in a text editor. (This feature is intended for knowledgeable users.)
• Run a simulation using data originally prepared for VS2DH 1.0. (This feature is intended for knowledgeable users.)

For information on running the postprocessor as a standalone application, see the postprocessor help pages by:

1. Launching the standalone postprocessor
2. Clicking the "Help" menu and then click "Postprocessor help".
