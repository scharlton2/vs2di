Overview

VS2DHI is a program that creates a visual-interactive environment for simulating fluid flow and energy transport in variably-saturated porous media.

VS2DHI consists of 3 components:

• a preprocessor for setting up a model simulation,
• a numerical model, VS2DH 3.3, for computing flow and transport,
• a postprocessor for visualizing simulation results.
