Tutorial: Output Options

In the output panel, "pressure head" is selected to be output to the main output file. Values of pressure head and temperature will be printed for every node at the selected output times and at the end of every recharge period.

Pressure heads and temperatures will be printed out at 1 day intervals and the mass balance will be printed to the output file only at requested output times.
