Changing the Value of a Contour

Before you begin, make sure that the appropriate data is selected in the "Active Data" Selector (see ).

1. Click the button.

2. "Select" the contour whose value is to be changed by click on it. (Click with the left mouse button)

3. Click the "Edit" menu and select "Contour Value..." The "Contour Value" dialog box is displayed.

4. Enter the new value of the contour into the dialog box.

5. Click "OK"
