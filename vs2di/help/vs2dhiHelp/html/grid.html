Laying Out the Grid

The grid consists of vertical and horizontal grid lines forming a rectangular array of cells. To lay out the grid, select "Grid" from the "Active Data" Selector (see ). The grid may be edited on the drawing area or on the Grid Window.

 • Create a uniform grid
Use the methods below to create a non-uniform grid.

 •  Subdivide a grid interval

Notes:

• The grid should be sufficiently large to cover the domain.
• The numerical model VS2DH (all versions) requires a ring of inactive border cells around the active grid. These border cells are automatically added by the preprocessor and are not shown or considered in the graphical display.
• When working in radial coordinates, the left edge of the grid must coincide with the z axis (r = 0).
