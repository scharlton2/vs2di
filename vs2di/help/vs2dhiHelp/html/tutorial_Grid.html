Tutorial: Grid Layout

To view the finite difference grid, select "Grid" from the "Show" menu. To manipulate the grid you must select "Grid" from the "Active Data" Selector (see ).

For this example, a uniform grid of 44 columns and 32 rows was set up by selecting "Uniform Grid" from the "Edit" menu.

To view or modify grid coordinates, display the grid window.

You may use the methods below to create a non-uniform grid.
 •  Move the grid
 •  Resize the grid
 •  Add grid lines
 •  Move a grid line
 •  Delete grid lines
 •  Subdivide a grid interval

This completes the set up of the simulation in the Tutorial. The example problem can now be run and results viewed in the postprocessor.
