Specifying Chemistry Classes

A chemistry class defines the initial water composition and the chemical reactants in a soil or porous medium in the modeled region.

Chemistry classes are contained in a table in the Chemistry Class window. If this window is not visible, display it by clicking the "Show" menu and selecting "Chemistry Class Window". If the window is in the reduced size, click "Expand".

The columns of the table contain seven chemistry class parameters.

There is always a default row which represents the "inactive" chemistry class (where reactions do not occur).

Cells in the table are not directly editable. Instead, use the buttons below the table to:
 •  Add a chemistry class
 •  Edit a chemistry class
 •  Delete a chemistry class
