Drawing a Chemistry Zone

A chemistry zone is drawn as a polygon.

Before you begin, make sure that "Chemistry Map" is selected in the "Active Data" Selector (see ).

1. If the Chemistry Class window is not visible, display it by clicking the "Show" menu and selecting "Chemistry Class Window". If the window is in its full size, you can reduce it by clicking "Shrink".

2. Click the  button.

3. Select a chemistry class by clicking the appropriate row in the Chemistry Class window. The properties of the selected chemistry class, including the color, will be assigned to the chemistry zone to be drawn next.

4. Draw the chemistry zone (polygon) by clicking on its vertices. (Click with left mouse button.)

5. Double click the last vertex.

6. Repeat steps 3 - 6 to draw additional chemistry zones.
