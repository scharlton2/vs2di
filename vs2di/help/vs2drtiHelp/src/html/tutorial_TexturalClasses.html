Tutorial: Textural Classes

To display the data for textural classes click the "Show" menu and select "Textural Class Window". If the window is in the reduced size, click "Expand".

There are 3 textural classes: the default inactive class, which is used to block out inactive zones within the domain, a sand, and a silt. The "Flow" tabbed panel displays the hydraulic properties and the parameters for the van Genuchten functions. The "Heat transport" tabbed panel shows dispersivities, heat capacities, and thermal conductivities. The "Solute transport" tabbed panel shows dispersivities and molecular diffusion coefficients.

To edit any of the values for a textural class highlight that row and click "Edit". Any of the parameters and the color of the class can then be modified. Once the changes are made, click "OK" to accept them or "Cancel" to ignore the changes.
