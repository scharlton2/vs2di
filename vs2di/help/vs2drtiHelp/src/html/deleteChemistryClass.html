Deleting a Chemistry Class

Note: Deleting a chemistry class will also delete corresponding chemistry zones in the chemistry map.

If the Chemistry Class window is not visible, display it by clicking "Show" and then selecting "Chemistry Class Window". If the window is in the reduced size, click "Expand".

In the Chemistry Class window:

1. Click (select) the row containing the chemistry class to be deleted. Note: You may not delete the "inactive" chemistry class.
2. Click "Delete". The "Confirm" dialog box is displayed.
3. Click "Yes" to confirm the deletion.
