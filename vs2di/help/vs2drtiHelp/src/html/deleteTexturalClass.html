Deleting a Textural Class

Note: Deleting a textural class will also delete corresponding textural zones in the textural map.

If the Textural Class Window is not visible, display it by clicking "Show" and then selecting "Textural Class Window". If the window is in the reduced size, click "Expand".

In the Textural Class Window:

1. Click (select) the row containing the textural class to be deleted. Note: You may not delete the "inactive" textural class.
2. Click "Delete". The "Confirm" dialog box is displayed.
3. Click "Yes" to confirm the deletion.
