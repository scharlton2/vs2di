Creating a Uniform Grid

A uniform grid has columns of equal widths and rows of equal heights. However, column width need not equal row height.

Before you begin, make sure that "Grid" is selected in the "Active Data" Selector (see ).

1. Click the "Edit" menu and select "Uniform Grid...". The "Uniform Grid" dialog box appears.

2. Enter the number of columns and rows

3. Select whether or not to shrink fit the grid around the domain.

4. Click "OK"
