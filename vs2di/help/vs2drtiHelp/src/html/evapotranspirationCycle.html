Specifying Evapotranspiration Cycle

Note: If neither evaporation (over bare soil) nor transpiration (plant root extraction) is simulated, this step may be skipped.

Note: The user is cautioned on the use of the evaporation and evapotranspiration options when heat transport is simulated by VS2DRT. These processes can influence and be influenced by soil temperature and available solar energy. As described in Lappala and others (1987) and implemented in VS2DRT, evaporation and evaoptranspiration are assumed to be isothermal. Users should evaluate the ramifications of this assumption in their applications. If these processes are an integral component of an application, then use of another numerical model that treats evaporation and evapotranspiration in a more realistic, nonisothermal fashion may be warranted.

Evapotranspiration (evaporation + transpiration) is simulated as a recurring cycle (see ). The cycle is composed of evapotranspiration periods of equal duration. Evapotranspiration parameters are defined at the beginning of each period, and are assumed to vary linearly from the beginning of one period to the beginning of the next period.

Evapotranspiration periods are contained in a table in the Evapotranspiration Window. If this window is not visible, display it by clicking the "Show" menu and selecting "Evapotranspiration Window". If this menu item is grayed out, you will have to first open the Model Options dialog box and select "simulate evaporation" and/or "simulate transpiration".

The columns of the table contains evapotranspiration parameters, which are divided into two panes:
 •  Evaporation (if evaporation is simulated.)
 •  Transpiration (if transpiration is simulated.)

Cells in the table are not directly editable. Instead, use the buttons below the table to:
 •  Set the evapotranspiration period length
 •  Add an evapotranspiration period
 •  Edit an evapotranspiration period
 •  Delete an evapotranspiration period
