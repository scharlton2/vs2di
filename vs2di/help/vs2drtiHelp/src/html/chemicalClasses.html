Specifying Chemical Classes

A chemical class refers to the water chemistry and the chemical reaction properties of a soil or porous medium in the modeled region.

Chemical classes are contained in a table in the Chemical Class Window. If this window is not visible, display it by clicking the "Show" menu and selecting "Chemical Class Window". If the window is in the reduced size, click "Expand".

The columns of the table contain 7 chemical class parameters.

There is always a default row which represents the "inactive" chemical class (where reactions do not occur).

Cells in the table are not directly editable. Instead, use the buttons below the table to:
 •  Add a chemical class
 •  Edit a chemical class
 •  Delete a chemical class
