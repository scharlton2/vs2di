Adding a Chemistry Class

If the Chemistry Class window is not visible, display it by clicking "Show" and then selecting "Chemistry Class Window". If the window is in the reduced size, click "Expand".

In the Chemistry Class window:

1. To add chemistry class n, click (select) row n - 1. The new chemistry class will be inserted after the selected row.
2. Click "Add". The "Add Chemistry Class" dialog box is displayed.
3. Click "Select Color". The "Select Color" dialog box is displayed.
4. Select a color by specifying either HSV (hue-saturation-value) or RGB (red-green-blue).
5. Click "OK". This closes the "Select Color" dialog box.
6. The chemical properties of each chemistry class are defined in a PHREEQC input file that must be created by the user outside of VS2DRTI. In the Add Chemistry Class window, the user can enter integer index values for the Solution and each of six chemical reactants, which correspond to indices in the PHREEQC input file. The index value must be greater than or equal to 0 for a Solution, and greater than or equal to -1 for all other reactants. A value of -1 indicates that no reactions of this type occur in this chemical zone. The user also has the option of selecting "As is" for each property. When a zone is mapped to cells, property values marked "As is" are not changed. If this property has been defined in a previous chemistry class, the previous value is retained. Selecting the "As is" option for all but one reactant allows the user to apply a reaction to a discrete area independent of all other solutions or reactions. Solution generally refers to the index number of a SOLUTION data block in the PHREEQC input file. Similarly, the index numbers for the other reactants generally refer to EQUILIBRIUM_PHASES, EXCHANGE, , GAS_PHASE, SOLID_SOLUTIONS, and KINETICS data blocks in the PHREEQC input file. It is also possible to refer to solutions and reactants that have been defined by SAVE or COPY data blocks in the PHREEQC input file.
7. Click "OK".
