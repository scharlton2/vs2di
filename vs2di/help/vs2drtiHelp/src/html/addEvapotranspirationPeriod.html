Adding an Evapotranspiration Period

If the Evapotranspiration Window is not visible, display it by clicking "Show" and then selecting "Evapotranspiration Window".

In the Evapotranspiration Window:

1. To add period n, click (select) row n - 1. The new period will be inserted after the selected row. Skip this step when adding the first period.

2. Click "Add". The "Add Evapotranspiration Period" dialog box is displayed.

3. Enter the evaporation parameters and/or the transpiration parameters.

4. Click "OK".
