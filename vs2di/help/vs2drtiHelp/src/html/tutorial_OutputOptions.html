Tutorial: Output Options

In the output panel, "pressure head" is selected to be output to the main output file. Values of pressure head will be printed for every node at the selected output times and at the end of every recharge period.

The mass balance will be printed to the output file only at requested output times. Observation point information will be printed to auxiliary output file "obsPoints.out" at every time step.
