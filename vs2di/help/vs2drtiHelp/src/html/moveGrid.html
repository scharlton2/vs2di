Moving the Grid

Before you begin, make sure that "Grid" is selected in the "Active Data" Selector (see ).

1. Click the  button.

2. Place the cursor inside the grid. (Avoid placing the cursor on edges or corners.)

3. "Drag" the grid to a new position by holding down the left mouse button and moving the mouse.

4. Release the mouse button when the grid is dragged to the desired new position.

Note: When working in radial coordinates, the grid can be moved only in the vertical (z) direction and not the horizontal (r) direction.
