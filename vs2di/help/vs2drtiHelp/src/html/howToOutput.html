How To Control VS2DRT Output

To write, at every time step, output variables (heads, saturation, concentrations, etc.) to the main output file (vs2drt.out):

1. Click the "Options" menu and select "Model..." The "Model Options" dialog box appears.
2. Click the "Output" tab.
3. Select the desired output variables.
4. Click "OK".
5. If the Recharge Period Window is not visible, display it by clicking "Show" and then selecting "Recharge Period Window". If the window is in the reduced size, click "Expand".
6. On the Recharge Period Window, check the box in the "print" column. (Two clicks are needed to switch from uncheck to check).

To write, at selected times, output variables to the main output file (vs2drt.out):

1. Click the "Options" menu and select "Model..." The "Model Options" dialog box appears.
2. Click the "Output" tab.
3. Select the desired output variables.
4. On the right side of the dialog box, specify the desired output times.
5. Click "OK".
6. If the Recharge Period Window is not visible, display it by clicking "Show" and then selecting "Recharge Period Window". If the window is in the reduced size, click "Expand".
7. Uncheck the box in the "print" column.

To write, at every time step, all balance components to the main output file (vs2drt.out):

1. Click the "Option" menu and select "Model..." The "Model Options" dialog box appears.
2. Click the "Output" tab.
3. For "Mass Balance" option, select "at every time step".
4. Click "OK".

To write, at selected times, all balance components to the main output file (vs2drt.out):

1. Click the "Option" menu and select "Model..." The "Model Options" dialog box appears.
2. Click the "Output" tab.
3. For "Mass Balance" option, select "at output times".
4. Specify the desired output times.
5. Click "OK".

To write, at every time step, selected balance components to file balance.out, and heads, temperatures, and concentrations at observation points to file obsPoints.out:

1. Click the "Options" menu and select "Model..." The "Model Options" dialog box appears.
2. Click the "Output" tab.
3. In the "Auxiliary Output Files" control group, select "at every time step".
4. Click the "Fluid Balance" tab.
5. Select the desired fluid balance components.
6. If simulating transport, click the "Energy/Solute Balance" tab, and Select the desired balance components.
7. Click "OK".

Note: For each selected component, 3 entries are written: total for simulation, total for time step, and rate for time step.

To write, at selected times, selected balance components to file balance.out, and heads, temperatures, and concentrations at observation points to file obsPoints.out

1. Click the "Options" menu and select "Model..." The "Model Options" dialog box appears.
2. Click the "Output" tab.
3. In the "Auxiliary Output Files" control group, select "at output times".
4. Specify the desired output times.
5. Click the "Fluid Balance" tab.
6. Select the desired fluid balance components.
7. If simulating transport, click the "Energy/Solute Balance" tab. and Select the desired balance components.
8. Click "OK".
