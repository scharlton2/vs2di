Note on van Genuchten Parameter "alpha"

The van Genuchten parameter "alpha" as implemented in this program is same as the one defined by van Genuchten (1980).

Here, "alpha" is the absolute value of the reciprocal of "alpha-prime" as used in VS2DT 2.5 and described in the program documentations by Lappala and other (1987) and Healy (1990).
