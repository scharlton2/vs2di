Editing a Chemistry Class

If the Chemistry Class window is not visible, display it by clicking "Show" and then selecting "Chemistry Class Window". If the window is in the reduced size, click "Expand".

In the Chemistry Class window:

1. Click (select) the row containing the chemistry class to be edited. Note: You may not edit the "inactive" chemistry class.
2. Click "Edit". The "Add Chemistry Class" dialog box is displayed.
3. To change color:
• Click "Select Color". The "Select Color" dialog box is displayed.
• Select a color by specifying either HSV (hue-saturation-value) or RGB (red-green-blue).
• Click "OK". This closes the "Select Color" dialog box.
4. Change the chemical properties.
5. Click "OK".
