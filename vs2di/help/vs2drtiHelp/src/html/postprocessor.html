Postprocessor

The postprocessor consists of a window (see ) to

• display simulation results
• control the simulation
• save simulation results on disk for later playback
• display fluid and solute balances

The postprocessor is interfaced with the numerical model VS2DRT in such a way that as the model runs, the simulated total hydraulic head, pressure head, moisture content, saturation, concentration, velocity, flux, temperature, and solute concentrations are displayed for each time step, thus creating a crude animation of model results. The speed of the animation depends on the computation time needed for each time step.

The Postprocessor may also be launched as a standalone application to:

• Play back a saved simulation. (This is the main intended use of the standalone postprocessor.)
• Run a simulation using data exported from the preprocessor, and possibly modified by the user in a text editor. (This feature is intended for knowledgeable users.)
• Run a simulation using data originally prepared for previous versions of VS2DH or VS2DT. (This feature is intended for knowledgeable users.)

For information on running the postprocessor as a standalone application, see the postprocessor help pages by:

1. Launching the standalone postprocessor
2. Clicking the "Help" menu and then click "Postprocessor help".
