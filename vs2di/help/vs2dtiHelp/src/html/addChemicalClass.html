Adding a Chemical Class

If the Chemical Class Window is not visible, display it by clicking "Show" and then selecting "Chemical Class Window". If the window is in the reduced size, click "Expand".

In the Chemical Class Window:

1. To add chemical class n, click (select) row n - 1. The new chemical class will be inserted after the selected row.
2. Click "Add". The "Add Chemical Class" dialog box is displayed.
3. Click "Select Color". The "Select Color" dialog box is displayed.
4. Select a color by specifying either HSV (hue-saturation-value) or RGB (red-green-blue).
5. Click "OK". This closes the "Select Color" dialog box.
6. The chemical properties of each chemical class are actually defined in a PhreeqC input file that must be created by the user outside of VS2DRTI. In this window, the user must enter integer index values for each of the 7 chemical properties which correspond to indices in the PhreeqC input data file. The index value must be greater than or equal to 0 for Solution, and greater than or equal to -1 for all other properties. A value of -1 indicates that no reactions of this type occur. The user also has the option of selecting "As is" for each property. When a zone with this option is mapped, property values marked "As is" are not defined and must be defined by mapping another zone that contains values for those properties. Selecting the "As is" option allows the user to easily map discrete areas where reactions occur.
