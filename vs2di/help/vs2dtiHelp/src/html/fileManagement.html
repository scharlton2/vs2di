File Management

Four types of files are used in VS2DRTI. They are:
 •  VS2 File
 •  VS2DRT Model Input Files
 •  VS2DRT Model Output Files
 •  Saved simulation Files

The relations between these files and the preprocessor, postprocessor, and VS2DRT are shown in this . All files are placed in a user-specified working directory. This is the directory in which the VS2 file is saved.

Under typical usage, the VS2 file is the most visible to the user because it contains the data entered into the preprocessor. By contrast, the model input files might be entirely transparent to many users because these files are automatically generated and requires no handling by the user.
