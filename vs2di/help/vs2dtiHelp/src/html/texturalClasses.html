Specifying Textural Classes

A textural class refers to the hydraulic and transport properties of a soil or porous medium in the modeled region.

Textural classes are contained in a table in the Textural Class Window. If this window is not visible, display it by clicking the "Show" menu and selecting "Textural Class Window". If the window is in the reduced size, click "Expand".

The columns of the table contain textural class parameters, which are divided into one, two. or three panes:
 •  Flow
 •  Heat transport (if heat transport is simulated)
 •  Solute transport (if solute transport is simulated)

There is always a default row which represents the "inactive" textural class (where hydraulic conductivity is zero).

Cells in the table are not directly editable. Instead, use the buttons below the table to:
 •  Add a textural class
 •  Edit a textural class
 •  Delete a textural class
 •  Import or export textural class data
