Displaying Simulation Results

There are five options for displaying simulation results.

Output Variable. The output variables are total head, pressure head, moisture content, saturation, concentration, and velocity or flux magnitude. You may display an output variable by selecting it in the drop-down list box next to the label "Display". You may switch the display from one variable to another when the model is not running, but you may not switch the display when the model is running.

Cell versus Contour. This option is selected in the drop-down list box next to the Output Variable box. When "Cell" is selected, the display consists of active cells each colored uniformly according to the color scale on the right. When "Contour" is selected, the display consists of contours with the regions between contours filled with a uniform color according to the color scale on the right. You may switch between the display between "Cell" and "Contour" when the model is not running, but you may not switch the display when the model is running.

Note: Displaying contours requires additional computation that will significantly slow down the simulation.

Color scale. The color scale represents values using a spectrum of colors from blue to red. To set the color scale:

1. Click the "Options" menu and select "Color Scale..." The Color Scale dialog box is displayed.

2. Enter the following 4 values:

     • Value at the red limit.
     • Value at the blue limit. Note that either red or blue may be used to represent the upper or lower limit.
     • Contour interval. Specifies the number of contour intervals. For example, if the value at the red limit is set to 100, the value at the blue limit is set to 0, the contour interval is set to 20, then 5 contour intervals will be displayed: 0-20, 20-40, 40-60, 60-80 and 80-100. This value has no effect when the display is set to "Cell".
     • Label interval. This controls the label frequency of the color scale.

3. Click "OK"

Velocity or Flux Vectors.

Velocity or flux vectors are drawn at cell centers as a line oriented in the direction of the vector and whose length is proportional to the vector magnitude.

If the bottom button on the toolbar displays , the vectors represent velocity. If the bottom button displays , the vectors represent flux (velocity times porosity).

To control how vectors are drawn, click the "Options" menu and select "Vector..." The Vector dialog box is display. Enter/select the following:

     • velocity or flux:Select the quantity represented by the vector.
     • scale: the magnitude represented by a vector one inch long.
     • column interval: A integral value of n means draw a vector every n-th column.
     • row interval: A integral value of n means draw a vector every n-th row.
     • show stem: Select this option if you desire a small black square at the start of the vector

Note:To display all vectors, enter 1 for both column interval and row interval.
