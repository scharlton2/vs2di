Specifying Recharge Periods

A recharge period is a time span during which boundary conditions (which include recharge rates) and stresses remain unchanged.

Recharge periods are contained in a table in the Recharge Period Window. If this window is not visible, display it by clicking the "Show" menu and selecting "Recharge Period Window". If the window is in the reduced size, click "Expand".

The columns of the table contain recharge period parameters, which define the duration of the recharge period and size of time steps.

Cells in the table are not directly editable. Instead, use the buttons below the table to:
 •  Add one or more recharge periods
 •  Edit one or more recharge periods
 •  Deleting one or more recharge periods
