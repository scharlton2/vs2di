Creating a New VS2 File

The data you enter to VS2DTI can be saved in a file known as a VS2 file. When you set up a new simulation, you should create a new VS2 file by clicking the "File" menu and selecting "New".

Note: If an existing simulation has been loaded in the preprocessor and the data have been changed, you will be prompted to save or discard the changes before creating the new VS2 file.

At any time while you are setting up the simulation, you may save the data by clicking the "File" menu and selecting "Save".

Notes:

• The first time you save a VS2 file, you will be prompted to specify its directory and name. The name if the VS2 file should terminate with the extension ".vs2".
• Each VS2 file should be saved in a separate, unique directory. This is known as the working directory. Additional files will be generated in this directory when you run a simulation.
