Haverkamp Model

For h < 0:
se = 1 / (1 + (h/alpha)beta)
Kr = 1 / (1 + (h/A')B')

For h >= 0:
se = 1
Kr = 1

where
se = effective saturation = (VMC - RMC)/(porosity - RMC)
VMC = volumetric moisture content
RMC = residual moisture content
Kr = relative hydraulic conductivity
h = pressure head
alpha = pressure head at which se = 0.5
beta = slope of the log-log plot of (1/se - 1) versus h
A' = pressure head at which Kr = 0.5
B' = slope of the log-log plot of (1/Kr - 1) versus h
