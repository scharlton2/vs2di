Selecting Model Options

Model options allow you to specify the flow and transport processes, computational algorithm, and simulation output. To select model options, click the "Options" menu and select "Model...". The "Model Options" dialog box is displayed.

Model options are divided into 7 categories with corresponding tabs in the dialog box:
 •  Basic
 •  Flow
 •  Transport (if transport is simulated)
 •  Solver
 •  Output
 •  Fluid Balance
 •  Solute Balance (if transport is simulated)
