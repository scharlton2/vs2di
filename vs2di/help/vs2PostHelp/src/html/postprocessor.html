Help for VS2DRTI Standalone Postprocessor

When the postprocessor is launched as a standalone application, it can operate in 2 modes. The type of file loaded to the postprocessor determines determines the mode in which the standalone postprocessor operates.

Playback Mode

When operating in playback mode, the postprocessor is used to play back a saved simulation. This is the main intended used of the standalone postprocessor.

Computation Mode

When operating in computation mode (intended for knowledgeable users), the postprocessor is used to run a simulation using either:

• data files exported from the VS2DRTI preprocessor and possibly modified by the user, or
• data files exported from the earlier VS2DTI or VS2DHI preprocessors or created manually for earlier versions of VS2DT or VS2DH. In this case, the simulation is run by Version 3.3 of VS2DT or VS2DH, not by VS2DRT.
