van Genuchten Model

For h < 0:
se = 1 / (1 + |alpha*h|beta)gamma
Kr = ( 1 - C * D (-gamma) )2 / D (gamma/2)
C = |alpha*h| (beta-1)
D = 1 + |alpha*h| beta

For h >= 0:
se = 1
Kr = 1

where
se = effective saturation = (VMC - RMC)/(porosity - RMC)
VMC = volumetric moisture content
RMC = residual moisture content
Kr = relative hydraulic conductivity
h = pressure head
alpha = parameter, see note below.
beta = parameter
gamma = 1 - 1/beta

Note: The van Genuchten parameter "alpha" as implemented in this program is same as the one defined by van Genuchten (1980). In other words, "alpha" is the absolute value of the reciprocal of "alpha-prime" as used in VS2DT 2.5 and described in the program documentations Lappala and other (1987) and Healy (1990).
