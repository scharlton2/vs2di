Computation Mode

The standalone postprocessor operates in computation mode when a namelist file (for example, vs2drt.fil) is loaded.

Note: The computation mode is intended for knowledgeable users who want to modify model data exported from VS2DRTI.

If you have exported data from the VS2DRTI preprocessor, the following 2 files are created:

• The namelist file (vs2drt.fil), which contains the names of input and output files to be opened by VS2DRT.
• The model data file (vs2drt.dat), which contains the input data to VS2DRT.

You may modify the model data file in a text processor.

In addition, model input files exported from the earlier VS2DTI or VS2DHI preprocessors or prepared for earlier versions of VS2DT or VS2DH may be used for running simulations in the standalone postprocessor. In this case, the simulation is run by Version 3.3 of VS2DT or VS2DH, not by VS2DRT.
