Help for VS2DI Standalone Postprocessor

When the postprocessor is launched as a standalone application, it can operate in 2 modes. The type of file loaded to the postprocessor determines determines the mode in which the standalone postprocessor operates.

Playback Mode

When operating in playback mode, the postprocessor is used to play back a saved simulation. This is the main intended used of the standalone postprocessor.

Computation Mode

When operating in computation mode, the postprocessor is used to run a simulation using either:

• data files exported from the VS2DI preprocessor, and possibly modified by the user, or
• data files originally prepared for VS2DT version 2.5 or VS2DH version 1.0, and having undergone partial migration or full migration.
Computation mode is intended for knowledgeable users.
