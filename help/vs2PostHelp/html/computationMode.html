Computation Mode

The standalone postprocessor operates in computation mode when a namelist file (for example, vs2dt.fil or vs2dh.fil) is loaded.

Note: The computation mode is intended for knowledgeable users who want to modify model data exported from VS2DTI or VS2DHI.

If you have exported data from the VS2DTI preprocessor, the following 2 files are created:

• The namelist file (vs2dt.fil), which contains the names of input and output files to be opened by VS2DT 3.3.
• The model data file (vs2dt.dat), which contains the input data to VS2DT 3.3.

Similarly, if you have exported data from the VS2DHI preprocessor, the following 2 files are created:

• The namelist file (vs2dh.fil), which contains the names of input and output files to be opened by VS2DH 3.3.
• The model data file (vs2dh.dat), which contains the input data to VS2DH 3.3.

You may modify the model data file in a text processor.

In addition, migration procedures are available so that model input files originally prepared for VS2DT 2.5 or VS2DH 1.0 may be used for running simulations in the standalone postprocessor.
