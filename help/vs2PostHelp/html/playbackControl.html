Controlling the Playback

If you have loaded a saved simulation file (for example, vs2dt.sim or vs2dh.sim), the postprocessor will be in "playback" mode. You can control the playback using the following buttons.

 Runs the playback. Saved results for the next time step is read from file and displayed, and this continues until the entire file is read.

 Stops the playback.

 Advances one time step. Saved results for the next time step are read from file and displayed.

 Restarts playback from beginning. Resets the playback to the start of the simulation.

If the animation is too fast, you may change the animation speed.
