Loading a Namelist File

To load a namelist file to the standalone postprocessor:

1. Click the "File" menu and then click "Load". A dialog box is displayed.
2. Navigate to the directory containing the namelist file and double click on the namelist file.

The first time you load the namelist file, you will have to:
 •  Set the display size and scale
 •  Set the color scales.

You may save the above settings to file as follows:
1. Click the "File" menu and then click "Save Settings". A dialog box is displayed.
2. Navigate to the desired directory, and enter the name of the file to store the settings.
3. Click OK.

The next time you load the same namelist file, you may restore the saved settings as follows:
1. Click the "File" menu and then click "Load Settings". A dialog box is displayed.
2. Navigate to the directory containing the settings file and double click on the file name.
