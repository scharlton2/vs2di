Tutorial: Initial Concentration

To view and/or modify the model initial concentration, select "Initial Concentration" from the "Active Data" Selector (see ).

Initial concentrations are specified by drawing contours of different values on the the screen. For the example problem, uniform initial values of 0. were desired. To accomplish this, a single closed contour was drawn around the entire domain. The value for the contour was set to 0. and the interpolation scheme used was "Step" rather than the default "Continuous" method. The color scale used for contour display can be customized by selecting "Color scale" from the "Edit" menu.
