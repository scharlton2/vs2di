Resizing the Grid

Before you begin, make sure that "Grid" is selected in the "Active Data" Selector (see ).

1. Click the  button.

2. Select the grid by clicking inside it with the left mouse button. A selected grid is surrounded by 8 small black squares (handles).

3. Place the cursor on one of the handles.

4. "Drag" the handle to a new position by holding down the left mouse button and moving the mouse. The grid will resize as the handle is dragged.

5. Release the mouse button when the grid is modified to the desired size.

Note: When working in radial coordinates, the left side of the grid is fixed at r = 0 and cannot be moved.
