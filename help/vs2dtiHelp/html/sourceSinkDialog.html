Source/Sink Properties Dialog Box

To change the flow rate and concentration of the source/sink point for a single recharge period:

• Click on the row for that period
• Change the values in the text boxes.For fluid injection, enter a positive flow rate, and a non-negative concentration. For withdrawal, enter a negative flow rate, and zero for concentration.
• Click "Change". (Don't forget this step or the values entered into the text boxes will not be recorded.)

To change the flow rate and concentration of the source/sink point for recharge periods A to B (inclusive):

• First click on row A. If necessary, scroll until row B is visible. Then hold down the Shift key and click on row B.
• If the flow rate is not constant during the selected periods, the words "as is" is displayed in the flow rate text box. The same applies to concentration.
• Change the values in the text boxes. For fluid injection, enter a positive flow rate, and a non-negative concentration. For withdrawal, enter a negative flow rate, and zero for concentration. If you leave the words "as is" in a text box, corresponding quantity will not be changed for the selected periods.
• Click "Change". (Don't forget this step or the values entered into the text boxes will not be recorded.)

Important Note: For cartesian (x-z) problems, the flow rate should be given per unit distance normal to the model domain. The flow rate should be in units of [L2/T].
