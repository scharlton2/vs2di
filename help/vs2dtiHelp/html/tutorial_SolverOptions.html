Tutorial: Solver Options

For the solver panel, the following values are selected: Relaxation parameter is set at its default value of 0.7. Minimum and maximum allowed iterations are set at 2 and 150, respectively. The simulation will be terminated if the solver of the matrix equation fails to converge. Maximum number of time steps is 1000. Closure criteria are 0.001 for the flow equation and 1.0E-5 for the transport equation.
