How to Set Up a Simulation

Setting up a simulation in the preprocessor involves 16 steps as listed in this section. New users are encouraged to follow these steps in the suggested order. Experienced users may wish to depart from the suggested order to better suit their preferences.

Go to first step: Create a new VS2 file

After setting up the simulation, you may

• run the simulation by invoking the postprocessor, or
• create model input files for VS2DT 3.3 without invoking the postprocessor. This is known as "exporting".
