Importing & Exporting Evapotranspiration Period Data

A user may wish to import a prefined set of ET Period data. A user can save or load these data using the "Custom" button on the Evapotranspiration window.

When you press the "Custom" button, a dialog box will appear. You have two options:

• Load Data From File: Pressing this button, you will be asked for a file name. The ET period data in the file will be inserted into the Evapotranspiration table. NOTE: this action will delete any ET period data previously entered to the Evapotranspiration table.
• Save Data To File: Pressing this button, you will be asked for a file name. All the current ET period data will be saved to the file.

The ET period data are stored in an ASCII text file and can be created or edited by the user.

If you want to examine a example data set, the simplest option is to run VS2DTI or VS2DHI, select the option to simulate evaporation and/or evapotranspiration (see Flow Options), create several ET periods, and save the data to file.

Notes:

• All solute-transport and energy-transport parameters can be stored in the same file, regardless of the type of simulation. Solute- and energy-transport parameters can be stored in the file even if neither transport method is currently being used.

File Format:

Notes:

• The same format is used regardless of whether evaporation, plant transpiration, or both evaporation and transpiration are simulated.
• There must be one line for each ET period, and each line must contain exactly 8 numbers: 3 evaporation parameters and 5 transpiration parameters.
• If only evaporation is simulated, then enter 0. for all transpiration parameters; similarly, if only plant transpiration is being simulated, then enter 0. for all evaporation parameters.
• There is no limit to the number of ET periods. 

Line format: PEVAL  SRES  HA  PTVAL  RD  RAbase  RAtop  HROOT
(separate values by one or more spaces)

where
PEVAL is potenital evaporation
SRES is surface resistance
HA is pressure potential of atmosphere
PTVAL is potential transpiration
RD is rooting depth
RAbase is root activity at bottom of root zone
RAtop is root activity at top of root zone
HROOT is pressure head in roots
