Specifying Source/Sink Points (Optional)

Note: This step is optional.

Source/Sink points denote points where fluid or mass enters the domain from a point. Inflow to the domain is indicated by a positive value. Outflow from the domain is indicated by a negative value.

Important Note: Source/sink points are allowed only for rectangular (x-z) problems. The flow rate should be given per unit distance normal to the model domain, and should be in units of [L2/T].

Select "Source/Sink Points" from the "Active Data" Selector (see ).

You may then:
 •  Create source/sink point
 •  Edit source/sink properties
 •  Move source/sink point
 •  Delete source/sink point
