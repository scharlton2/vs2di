VS2DH 3.3 Model Output Files

Model output files contains text (ASCII) output generated by the numerical model VS2DH 3.3. These files include:

• A main output file with labels and headings, and is therefore suitable for viewing (in printed form or with a text editor), and
• Several auxiliary output files that contain numerical values without labels and heading, and are more suitable for importing into graphing programs.

When running a simulation using the postprocessor, model output files are put in the current working directory (the directory where the VS2 file is stored). The names of output files are:

• vs2dh.out: main output file
• variables.out: auxiliary output file containing heads and temperatures
• balance.out: auxiliary output file containing selected mass balance components
• obsPoints.out: auxiliary output file containing heads, temperatures, etc. for selected observation points
• file07.out: this file is currently not used. Although it is generated, it is always empty.
