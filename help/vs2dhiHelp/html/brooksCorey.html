Brooks-Corey Model

For h < h b :
se = (h / h b ) (-lambda)
Kr = (h / h b ) (-2-3*lambda)

For h >= h b :
se = 1
Kr = 1

where
se = effective saturation = (VMC - RMC)/(porosity - RMC)
VMC = volumetric moisture content
RMC = residual moisture content
Kr = relative hydraulic conductivity
h = pressure head
h b = bubbling or air entry pressure head
lambda = pore size distribution index
